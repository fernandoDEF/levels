Crear una level y retornar su nombre:

mutation {
  createLevel(level: {
    name: "user", 
    level: "level-0",
    progress: 0.0,
    enable: false
  }) {
    name
  }
}


Actualizar la level con id: 1, y retornar su id, nombre usuario y nivel:

mutation {
  updateLevel(id: 1, level: {
    name: "user 2",
    level: "level-0",
    progress: 0.0,
    enable: false
  }) {
    id
    name
    level
  }
}
